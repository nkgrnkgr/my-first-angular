## 試し方

### はじめに

clone したらインストール

```
npm install
```


### Reactのビルド

Reactのディレクトリに移動してビルド

```
cd my-1st-angular/react-component
npm install
npm run build
```

### Reactのサーブ

Angularのディレクトリで

```
npm run serve:react
```

### Angular の起動

```
npm run start
```

### ブラウザアクセス


http://localhost:4200/
